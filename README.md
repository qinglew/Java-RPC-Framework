# Java-RPC-Framework

This is a simple Java RPC framework.

## Functions

* Network
  * Socket
  * Netty
* Serializer
  * JDK
  * Hessian2
  * Kyro
  * Json
* Self-Defined Protocal
* Automatic Server Registry and Dregistry
  * Nacos
* Automatic Service Scan
* Load Balance
  * Random
  * Round Robin
